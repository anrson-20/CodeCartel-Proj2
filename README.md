# CodeCartel-Proj2
ELEC1005 Project 2
AidSync is an app for volunteer to rescue the victims. It is designed to connect patients with nearby trained volunteers and emergency resources. The platform can greatly reduce emergency response time by providing SOS alerts, real-time communication, GPS, volunteer skill matching and equipment transportation. 

Features
Relevant informationabout the user and volunteer gets collected when they sign up. This facilitates suitable treatment by suitable volunteers.
User requests for help within two taps through their preferred mode of communication.
Volunteers are grouped and training sessions are scheduled according to their skill sets and experience

## Quick Start
1. Clone the repository
2. Open the powerapps
3. Import the Aid "Sync.msapp" file into your environment
4. Run the app
5. Navigate to Edit
