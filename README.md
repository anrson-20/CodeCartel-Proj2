# CodeCartel-Proj2
ELEC1005 Project 2 - AidSync

AidSync is an app for volunteers of an organization to locate and help patients in need of medical assistance. It is designed such that patients can quickly raise an alert and recieve medical treatment through nearby trained volunteers and emergency resources. 

Features:
- SOS Help button for patients
- Map showing live coordinates of both patients and volunteer using GPS
- Equipment list generated to assist volunteer in high pressure situations
- Emergency log for history of events and details

How to use:
1. Download Aid Sync Finl.msapp from the repository
2. Open Microsoft PowerApps and click on import on the top menu bar
3. Select the downloaded file. You will be taken to the Welcome Screen

Common Confusion:
Difference between Sign Up and Sign In : Sign Up is for creating an account for the first time, while sign in is for logging in to an existing account
